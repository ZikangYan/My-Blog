# 我的博客
我的研究生阶段研究或者说是打工阶段，干过许多的活，包括但是不限于3D视觉相关工作、物理信息神经网络，大模型评测，大模型部署RAG等，行人属性识别PAR，数据集的拍摄标注和组织还参与过工业的异常检测。但是零零总总的我总觉得我又陷入了一种”碌碌无为“感觉。觉得啥也没有学到啥都没有学明白，然后稀里糊涂的又选择了读PHD，总之对未来的发展没有一个清晰的认识。我又选择了硕博连读，但是我没有清楚的认知，有些东西我感觉上是我想做的，但是又没有做成功，所以我决定开始，从今天开始记录并分享生活。通过使得自己被看到，从而监督自己的日常。

如果我能够成功，这就是一份从0开始的发展模板，如果我不能够成功，这就是一份失败的模板。希望能够通过这个，能够让我真正的觉得我做的很踏实，真正的找到我所喜欢的，我能够自信的！

目前我的博客主要包含三个方面：
1. 读的文章--收集和总结
2. 学习--在科研工作之外的，自己想学习的技术
3. 日常--日常工作总结
4. 我自己的工作
